# Inventory Management System

A beginner-friendly Inventory Management System built using Python.

## Features

- Add Product
- Update Product
- Delete Product
- Search Product
- Display Inventory
- Sell Product
- Restock Product
- Inventory Value Calculation
- Revenue Calculation
- Low Stock Report
- Out of Stock Report
- Search by Price Range

## Technologies Used

- Python
- Jupyter Notebook

## Concepts Used

- Dictionary
- Nested Dictionary
- Loops
- Functions
- Conditional Statements
- CRUD Operations
