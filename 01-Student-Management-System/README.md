# 🎓 Student Management System

A beginner-friendly Student Management System built using Python. This project demonstrates how to manage student records using dictionaries, loops, and conditional statements.

##  Features

- Add Student
- Search Student
- Update Student Details
- Delete Student
- Display All Students
- Dictionary-Based Data Storage

## 🛠️ Technologies Used

- Python
- Jupyter Notebook

## 📚 Concepts Covered

- Variables
- Input & Output
- Conditional Statements
- Loops
- Dictionaries
- Nested Dictionaries
- CRUD Operations

## How to Run

1. Open the notebook in Jupyter Notebook or VS Code.
2. Run all cells.
3. Follow the menu displayed on the screen.
